# china-main-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不针对任何特定域名或网站，仅作为内容存档与展示使用。

## 目录说明

- `pages/` — 存放所有归档的 HTML 页面文件，每个文件为一个独立页面
- `assets/` — 页面所需的公共资源文件，如样式、图片、脚本等
- `index.html` — 仓库根目录下的入口页面，可提供导航或索引功能

## 页面归档说明

归档的页面均为独立 HTML 文件，无需后端支持，可直接在浏览器中打开查看。文件命名尽量反映内容主题，便于检索和引用。

页面内容可能来自不同来源，经过整理后统一存放于此。本仓库不保证所有页面的持续可用性，也不对页面内容的准确性或时效性作出承诺。

## 使用方式

您可以通过以下方式使用本仓库中的页面：

1. 直接访问 GitHub Pages 部署的站点（如果已启用）
2. 克隆仓库到本地，用浏览器打开任意 HTML 文件
3. 自行部署到静态托管服务
